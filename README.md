# sr-vibe-coding

This is a vibe coding workshop for Serious Rooster. It shows one of the many ways to use vibe coding for development purposes.
Try the steps below and try to reuse the agents.


## first question to ask
Read request.txt and come up with some additional ideas which can be added to the website. Save the ideas in /docs/IDEAS.md

Do not build anything, provide ideas only.


## second question to ask
Read request.txt and all ideas given in /docs/IDEAS.md. Create and write the agents as specified in request.txt.


## Third question to ask
Make a plan on how to build this website using the given agents, and write it to PLAN.md. Do not build anything.


## Fourth question to ask
Start building the website accordin PLAN.md and do it phase by phase. After each phase, ask my permission to go ahead. Make use of the provided agents.
